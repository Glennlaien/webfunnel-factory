# result pages

Result pages bridge onboarding into monetization.

Canonical result page types:

- `summary_page`
- `analysis_page`
- `plan_generation_page`
- `plan_ready_page`

## Design Expectations

- Must feel personalized
- Should reference answers from earlier pages
- Should build trust before paywall
- Must follow `global-rules/compliance-rules.json`

## Required Before Paywall

Most paid funnels should include:

```text
summary_page
-> analysis_page or plan_generation_page
-> plan_ready_page
-> paywall_page
```
