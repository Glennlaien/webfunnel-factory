# UI Style Recipe Presets

These presets are reusable global visual systems for Web2App funnels.

Each preset follows `contracts/ui-style-recipe.contract.json` and can be selected by the UI Designer agent before generating:

- `outputs/design/ui-style-recipe.json`
- `outputs/design/theme.json`
- `outputs/design/art-direction.json`
- `outputs/design/screen-blueprints.json`

## Presets

| Recipe | Best For | Core Feeling |
| --- | --- | --- |
| `hard_training` | Male training, military fitness, calisthenics, strength, muscle building | disciplined, direct, strong |
| `calm_wellness` | Tai chi, yoga, seniors, low-impact movement, recovery | calm, safe, spacious |
| `energetic_fitness` | Women's weight loss, toning, young active fitness | bright, motivating, energetic |
| `clinical_trust` | Health management, BMI, recovery, pain support, credibility-heavy funnels | clean, trustworthy, measured |
| `lifestyle_companion` | Habits, light movement, daily routine, long-term lifestyle support | warm, friendly, sustainable |

## Selection Rule

The UI Designer should select one preset using product strategy evidence, then make only product-specific overrides that preserve the preset's global visual system.

Do not mix unrelated colors, fonts, button styles, or card radius systems page by page.
