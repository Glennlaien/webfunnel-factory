# Height And Weight Input Recipe

Use numeric inputs with segmented unit tabs.

## Height

Supported units:

- `cm`
- `ft_in`

For `cm`, render one numeric input.
For `ft_in`, render two numeric inputs: feet and inches.

Switching units recalculates the current value immediately.

## Weight

Supported units:

- `lb`
- `kg`

Render one numeric input and a segmented unit control.

Switching units recalculates the current value immediately.

## Layout

Mobile-first layout. Desktop uses centered wide Web funnel content, not a phone mockup.

Do not add BMI explanation cards unless explicitly requested by product requirements.
